# CalendarSwing_JavaProject

개인 맞춤형 가계부 및 소비패턴 분석 시스템

달력 기반의 소프트웨어를 통해 사용자는 수입/지출 내역 가계부를 작성할 수 있다. 

아래는 소프트웨어의 주요 기능들이다. 

- 월별, 연도별 소비 내역 카테고리별 분석 및 그래프 산출
- 사용자 평균과 내 지출 내역 비교
- 목표치를 설정하여 관리
- 월별 고정 수입,지출 내역 관리
- 한달 수입, 지출 내역을 EXCEL파일로 출력


이클립스로 실행 권장
1. 깃허브 저장소 URL 복사: 가져오고 싶은 깃허브 저장소 페이지에서 초록색 Code 버튼을 클릭하고 URL을 복사
2. 터미널 열기: 컴퓨터에서 터미널 또는 명령 프롬프트(cmd)를 실행 
3. 프로젝트를 저장할 폴더로 이동: cd 명령어를 사용하여 코드를 저장할 폴더로 이동 (예: cd Documents/Projects) 
4. git clone 명령 실행: 터미널에 git clone [복사한 URL]을 입력하고 실행. [복사한 URL] 부분에는 복사한 깃허브 저장소 주소를 붙여 넣으면 됨.
